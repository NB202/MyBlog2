# MyBlog2

android大作业，本项目包含三个基本功能，分别是博客爬虫，缓存服务，本地随笔。
博客爬虫是分页爬取博客园博客，在app上显示博客的html格式（包括博客中的图片）；
缓存服务是按策略缓存一些博客到本地数据库，当用户没网时打开app也能有博客可浏览，而不是简单的请求连接网络提示，不会影响用户体验；
本地随笔是用户可以在浏览博客的时候记笔记或者写一些随笔，保存在本地数据库，用户可以方便的浏览，修改这些随笔。
